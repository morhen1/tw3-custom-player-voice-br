# Contribuindo

Relatos de falha devem informar, quando possível:

- ID hexadecimal da fala;
- missão, personagem e local do diálogo;
- texto exibido na legenda;
- tipo do problema: silêncio, corte, pronúncia, concordância, volume ou sincronia;
- versão do jogo e dos mods relacionados.

Não anexe ao GitHub falas extraídas do jogo, gravações privadas, modelos de voz,
WAV/WEM, `brpc.w3speech` ou dados pessoais. Um trecho curto gravado dentro do
jogo só deve ser compartilhado quando necessário e permitido pela plataforma.

Mudanças na voz ou no consentimento não são aceitas por pull request. Mudanças
de código devem manter os testes existentes e incluir novos testes quando
alterarem comportamento verificável.
