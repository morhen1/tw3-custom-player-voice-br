# Custom Player Voice BR — voz feminina PT-BR

## Descrição curta

Adiciona uma voz feminina em português brasileiro para as falas normalmente
associadas ao Geralt, destinada ao uso com personagens femininas do mod Custom
Player Characters em *The Witcher 3* 4.04 para PC.

## O que o mod faz

- substitui 19.359 falas de Geralt em português brasileiro;
- preserva 17 entradas oficiais silenciosas ou não verbais;
- mantém os dados CR2W originais de sincronização;
- cobre jogo-base, Hearts of Stone, Blood and Wine e DLCs oficiais;
- usa WEM Opus 48 kHz, com canais compatíveis com os originais.

Importante: a substituição é global. Enquanto este mod estiver ativo, o Geralt
original também usará a voz feminina. Desative o mod para voltar à dublagem
normal.

## Requisitos

- The Witcher 3 versão 4.04 para PC;
- vozes configuradas em português brasileiro;
- Custom Player Characters instalado separadamente.

## Instalação manual

1. Extraia o arquivo baixado.
2. Copie `modCustomPlayerVoiceBR` para a pasta `The Witcher 3\Mods`.
3. Confirme que existe:
   `The Witcher 3\Mods\modCustomPlayerVoiceBR\content\brpc.w3speech`.
4. Inicie o jogo e selecione português brasileiro para as vozes.

## Desinstalação

Remova somente a pasta:

```text
The Witcher 3\Mods\modCustomPlayerVoiceBR
```

## Compatibilidade

Não use simultaneamente com outro mod que substitua `brpc.w3speech` ou as falas
de Geralt em português brasileiro.
